# meteor-react-mui5-template

This is a starter for a Meteor + React + Material UI 5 project. It provides all user account logic and provides
full i18n support.

## Features

- Meteor 2.14
- React 18.2
- React router 6.21
- Mui 5.15
- React-i18n (use of i18n on server and client)

## Run the app

    meteor npm start
    
## Testing

    meteor npm test

## Linting

    meteor npm lint
